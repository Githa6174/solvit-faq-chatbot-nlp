# SolvIT Intelligent FAQ Chatbot (NLP Project) 🤖

Sistem Tanya Jawab (Chatbot) otomatis untuk layanan IT Support SolvIT Bali. Menggunakan teknik NLP dasar untuk menghitung kemiripan vektor antara pertanyaan pelanggan dengan basis pengetahuan perusahaan.

## Konsep NLP
- **Preprocessing:** Cleaning, Stopword Removal, & Stemming (Sastrawi).
- **Representasi Teks:** TF-IDF & Word2Vec.
- **Similarity Measure:** Cosine Similarity.

## Cara Penggunaan
1. Install dependencies: `pip install -r requirements.txt`
2. Jalankan notebook `Chatbot_FAQ_SolvIT.ipynb`.
3. Masukkan pertanyaan pada fungsi `chatbot_solvit()`.
